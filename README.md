# تطبيق الجدولة الذكي

تطبيق Android متقدم لإدارة الأحداث والتذكيرات بلغة Kotlin و Jetpack Compose.

## المميزات

- 📅 التقويم الشهري التفاعلي
- ➕ إضافة وتعديل الأحداث بسهولة
- 🔔 نظام التذكيرات المحلية
- 💾 قاعدة بيانات محلية (Room)
- 🎨 واجهة عربية احترافية
- ⚡ أداء عالي واستهلاك بطارية منخفض

## المتطلبات

- Android 7.0+ (API 24)
- Android Studio Giraffe أو أحدث
- JDK 11+

## البناء

```bash
./gradlew assembleDebug
```

## التثبيت

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```
